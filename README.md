# Calculator
**By Subaita Sajjad · Personal Project**

A fully functional GUI calculator built from scratch with Python and Tkinter.

---

## About
This is a working desktop calculator application built using Python's built-in Tkinter library. The user can perform basic arithmetic operations through a clean button interface. Built to practice GUI development, event handling, and working with Python's standard library.

---

## Features
- Addition, subtraction, multiplication, and division
- Clear button (C) to reset the input field
- Error handling for invalid expressions
- Clean grid-based button layout
- Fully functional display field showing input and results in real time

---

## Tech Stack
- Python 3
- Tkinter (GUI)
- Event-Driven Programming
- Grid Layout Management

---

## How to Run

```bash
# No installs needed — Tkinter comes built into Python

# Run the calculator
python Calculator.py
```

---

## Controls

| Button | Action |
|---|---|
| 0 – 9 | Enter numbers |
| + − * / | Arithmetic operators |
| = | Calculate and display result |
| C | Clear the input field |
